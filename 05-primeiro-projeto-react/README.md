## Descrição do projeto

<p align="justify">
Nessa etapa daremos início ao aprendizado do ReactJS, criando o primeiro projeto já utilizando TypeScript. Nesse ponto será necessário o conhecimento dos conceitos de componente, estado e propriedades.

Nas Estrutura e padrões temos os fundamentos que serão utilizados tanto na web (com o ReactJS) quanto no mobile (utilizando React Native), como padronização de projetos React, estrutura de pastas, estilização no ReactJS e React Native, rotas e consumo de API's externas. Quando criamos a aplicação, estamos aptos a criar rotas, configurar a dependência responsável por lidar com as rotas na aplicação, onde define qual página será mostrada em tela, qual endereço deve ser exibido na barra do navegador, etc.

Quando utilizamos Styled Components que é a forma de fazer as nossas estilizações como se fossem componentes customizados e fazendo as estilizações globais do primeiro projeto. O mais legal é poder aplicar esse pacote tanto em projetos ReactJS quanto em projetos React Native sem mudar praticamente nada.

Quando estilizamos o Dashboard, mexemos na parte estrutural e visual da primeira tela do GitHub Explorer que é o Dashboard, aplicando mais conceitos do styled-components, percebe-se na prática como funcionam as estilizações que se aplicam apenas em um componente.

No decorrer do projeto vamos conectar a API, tratar os erros, navegar entre rotas, estilizar pequenos detalhes e listar issues da API, que é exibir as informações reais da API como o numero de stars, forks e issues abertas.
</p>

## Instalações e Dependências

:boom: yarn create react-app 05-primeiro-projeto-react --template=typescript -> Criando as pastas

:boom: yarn install -> Criar a pasta do node_modules caso você faça o clone do projeto

:boom: yarn start -> rodar o servidor

:boom: http://localhost:3000 -> Endereço do servidor

:boom: yarn add -D eslint-plugin-react@^7.21.5 @typescript-eslint/eslint-plugin@latest eslint-config-airbnb@latest eslint-plugin-import@^2.22.1 eslint-plugin-jsx-a11y@^6.4.1 eslint-plugin-react-hooks@^4 @typescript-eslint/parser@latest

:boom: yarn add react-router-dom

:boom: yarn add @types/react-router-dom -D

:boom: yarn add styled-components

:boom: yarn add @types/styled-components -D

:boom: vscode-styled-components

:boom: yarn add polished

:boom: yarn add react-icons

:boom: yarn add axios
